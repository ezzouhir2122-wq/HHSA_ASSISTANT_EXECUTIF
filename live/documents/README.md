# live/documents/

Sorties opérationnelles actives — documents générés par les workflows.

Les PDFs finaux vont dans `exports-pdf/`. Les sources (Markdown, JSON) restent dans leur dossier de type.
Une fois un document livré au client, copier dans `clients/[nom]/[type]/`.

| Dossier | Contenu |
|---------|---------|
| `propositions/` | Propositions générées (sources) |
| `devis/` | Devis générés (sources) |
| `factures/` | Factures générées (sources) |
| `contrats/` | Contrats en cours de rédaction |
| `exports-pdf/` | Tous les PDFs produits — format de livraison final |
