# clients/

Un dossier par client. Structure standard définie dans `_template/`.

## Utilisation

Pour un nouveau client :
1. Dupliquer `_template/` → renommer avec le nom du client (ex: `foster-marsh/`)
2. Déposer tous les documents liés à ce client dans les sous-dossiers correspondants

## Structure par client

| Dossier | Contenu |
|---------|---------|
| `propositions/` | Propositions commerciales envoyées |
| `devis/` | Devis émis |
| `factures/` | Factures émises |
| `contrats/` | Contrats signés ou en cours |
| `livrables/` | Livrables remis au client |
