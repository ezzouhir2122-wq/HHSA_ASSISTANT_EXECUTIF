Architecture — Gestion des stocks

Approche A : WhatsApp + Google Sheets + Dashboard web

ENTRÉE
📱 WhatsApp
Staff envoie un message en français ou darija
"entrée 10kg poulet" · "dkhel 5 bouteilles zit"
↓ webhook
CERVEAU AGENTIQUE
🤖 Claude API
Extrait : action (entrée/sortie) · produit · quantité · unité

↓ write
STOCKAGE
📊 Google Sheets
3 feuilles : Produits (catalogue + seuils) · Mouvements (log) · Stock Actuel (calculé)
↙ read     ↘ alert
SORTIE — GÉRANT
🖥 Dashboard web
Vue stock temps réel · historique · indicateurs rouge/jaune/vert
ALERTES
📱 WhatsApp gérant
Alerte si stock < seuil · résumé quotidien automatique
